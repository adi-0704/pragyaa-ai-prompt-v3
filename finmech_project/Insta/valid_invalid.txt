Your Task
1.	Identify if the call is valid for evaluation.
2.	Generate a short justification explaining why the call is valid or invalid.
________________________________________
Call Validation Criteria
✅ DEFAULT: Mmark as invalid if the call is clearly and obviously empty or contains ONLY automated responses.
✅ Mark as VALID if:
•	There is ANY proper conversation between agent and customer (even if brief, one-sided, or minimal).
•	There is ANY meaningful interaction, even if the customer responds minimally.
•	When in doubt, mark as VALID.
________________________________________
❌ Mark as INVALID if the call is ABSOLUTELY CLEAR that:
•	The call is completely Empty — no content at all, completely blank audio, or completely silent from start to end.
•	The call contains ONLY automated responses — ONLY IVR prompts, ONLY system messages, or ONLY automated voice with ZERO human conversation from start to end.
If there is an response from customer initially then it should not be invalid
________________________________________
Evaluation Rules
•	BE LENIENT: Default to marking calls as VALID. Only mark as invalid if it's absolutely clear the call is empty or contains ONLY automated responses.
•	Mark the call as VALID if there is ANY human conversation, speech, or dialogue between agent and customer, even if:
  - Brief or minimal
  - One-sided (only agent speaking)
  - Customer responds with just "yes", "no", "okay", or similar minimal responses
  - Ends with customer silence
  - Ends with automated disconnection messages
•	If the call had ANY conversation earlier but later ends with customer silence or automated disconnection messages, mark it as VALID. The presence of automated messages at the end does NOT make it invalid if there was ANY real conversation earlier.
•	Mark the call as INVALID ONLY if it's ABSOLUTELY CLEAR that:
  - The call is completely empty (no content, blank, silent) from start to end with ZERO speech.
  - The call contains ONLY automated responses (IVR, system prompts, no human conversation at all) from start to end with ZERO human speech.
•	CRITICAL: When in doubt, mark as VALID. A call with ANY human speech or conversation is valid, regardless of how it ends or how brief it is.
