If MET then Score = 5
RPC (Right Party Customer) is considered VERIFIED only if BOTH are present:
1) Agent explicitly asks for / states the customer name (RPC) in the greeting segment.
2) Customer explicitly confirms the same name (by replying with the name or confirming "yes/correct/haan" to that specific name).
Agent must Start call with greeting within 3 to 5 seconds. (Check the Time-stamp in the Transcript to decide whether the agent started introducing in first 3-4 seconds)

Strictly Check and confirm the Right Party Customer’s Name. If either (1) or (2) above is missing OR the agent starts pitching before RPC confirmation, then assign Greeting_RPV_Eng = 0.
👉 Only explicit RPC name confirmation counts. Generic address like “sir/ma’am”, “hello ji”, “madam ji” WITHOUT asking/confirming the customer name DOES NOT count as RPC confirmation.
👉 If the agent assumes identity from previous context/application details BUT does not explicitly ask/state and get customer confirmation of the customer name in THIS call, it is treated as NOT confirmed.
👉 If agent says a name (e.g., "Rahul ji") but the customer does not confirm the SAME name (silence/unrelated reply/busy/callback), then Greeting_RPV_Eng = 0.
Examples:
•	“Good morning, I’m Robin Calling on behalf of ICICI Bank. Rahul ji, baat kar rahe ho?” (Agent asks)
  “Yes, Rahul ji.” (Customer confirms)
•	“Hello, Good afternoon. Is this Rahul ji?” (Agent asks)
  “Yes, correct.” (Customer confirms)
•	“Hello, Good Morning. App Rahul ji hi hain na?” (Agent asks)
  “Haan ji.” (Customer confirms)
•	“Good morning, I’m Robin calling on behalf of ICICI Bank.” (Agent does not ask customer name)
  “Yes/Ok.” (No explicit customer confirmation of any RPC name)
  (Greeting_RPV_Eng = 0)
•	“Good morning, Rahul ji.” (Agent states a name, but does not get explicit confirmation)
  “Hello?” / “Who is this?” (No explicit “Yes Rahul ji/correct/haan” confirmation)
  (Greeting_RPV_Eng = 0)
•	“Hello, Good afternoon. Is this Rahul ji?” (Agent asks)
  “Busy, call later.” / silence (Customer does not explicitly confirm the asked name)
  (Greeting_RPV_Eng = 0)
