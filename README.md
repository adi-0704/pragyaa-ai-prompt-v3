# Pragyaa.AI — Prompt Evolution Engine

> OpenClaw/Hermes powered audit prompt optimizer for ICICI Bank DC verification pipeline.

## What it does

1. **Upload** an AI vs Verifier audit Excel file (.xlsx)
2. **Analyze** root causes of discrepancies between AI and human verifier judgments
3. **Generate** an optimized audit prompt with data-driven calibration

## Key Features

- 🔬 Client-side Excel parsing (no server needed)
- 📊 Root cause heatmap with parameter-level failure analysis
- 🔒 3-Tier consent system calibrated to Hindi sales call patterns
- 💰 Content-based charges evaluation (not speed-based)
- 📋 One-click copy/download of optimized prompt
- 📈 Evolution history tracking across successive feedback cycles

## Tech Stack

- Vanilla HTML/CSS/JS (zero build step)
- [SheetJS](https://sheetjs.com/) for browser-based Excel parsing
- Deployed on Vercel
